Not A Normal Field Trip

This is my demo version for the game Not A Normal Field Trip that is in development. You can find it here at MakMantisGames

Description and Applications Used

It is a topdown adventure/puzzle 2d game made using Godot. I made all the artwork for it using Illustrator besides the main menu screen (see to the left) which I designed in Photoshop.

Process

I normally start all my projects by writing down key points/characters/ideas I want to implement and then sketching how I want it to look. Here is the final sketch of what I decided my main level would look like, and then the final version after I put it into Illustrator and used it as a template to draw overtop of.

I designed Eino, the character you play as, also in Illustrator. But then I imported it into Photoshop to create this sprite sheet.
